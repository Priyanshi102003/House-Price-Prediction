# 🏠 House Price Prediction

## Overview

This project implements a Machine Learning model to predict house prices based on features such as house size, number of bedrooms, and age of the property. The model is trained using a Linear Regression algorithm to estimate house prices from the provided input data.

The project demonstrates a basic machine learning pipeline including data loading, model training, model persistence, and prediction.

---

* pandas
* scikit-learn
* Joblib
